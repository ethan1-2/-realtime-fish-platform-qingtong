# 场景 3: SAAS 平台计费与配额治理

## 目标

补齐多租户治理能力：

- 按租户计费（账单口径清晰，争议可追溯）
- 配额与隔离（保护平台稳定）
- 幂等与对账（重试/补单/重放不重不漏）

## 输入与输出（建议）

- 输入：
  - 支付与风控相关事件（可复用场景 1/2）
  - 配额配置/变更
- 输出：
  - Doris: 租户日账单聚合（至少 1 张表）
  - Kafka: `tp_tenant_throttle`（超限控制指令）
